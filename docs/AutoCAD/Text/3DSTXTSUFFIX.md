# 3DSTXTSUFFIX

## Description

Adds a suffix to the selected **TEXT/MTEXT** objects.

## Usage

* Run command (3DSTXTSUFFIX)
* Enter a suffix value
* Select **TEXT/MTEXT**

## Example Output

```
Command: 3DSTXTSUFFIX
3DS> Enter suffix text: test
3DS> Select text entities: 1 found
3DS> Select text entities:
```
