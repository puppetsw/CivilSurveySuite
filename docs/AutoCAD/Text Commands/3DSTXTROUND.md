# 3DSTXTROUND

## Description

Rounds text numbers to a specified decimal place.

## Usage

* Run command (3DSTXTROUND)
* Select **TEXT/MTEXT**
* Enter the number of decimal places

## Example Output

```
Command: 3DSTXTROUND
3DS> Select text entities: 1 found
3DS> Select text entities:
3DS> Enter number of decimal places: 2
```
