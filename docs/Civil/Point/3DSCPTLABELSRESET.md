# 3DSCPTLABELSRESET

## Description

Resets the attached label of any selected **CogoPoints**.

## Usage

* Run command (3DSCPTLABELSRESET)
* Select **CogoPoint(s)**

## Example Output

```
Command: 3DSCPTLABELSRESET
3DS> Select CogoPoint labels to reset: 1 found
3DS> Select CogoPoint labels to reset:
```
